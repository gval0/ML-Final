# Walmart рЃЏрЃљрЃдрЃљрЃќрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљ

## Kaggle-рЃўрЃА рЃЎрЃЮрЃюрЃЎрЃБрЃарЃАрЃўрЃА рЃЏрЃЮрЃЎрЃџрЃћ рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА Kaggle-рЃўрЃА рЃЎрЃЮрЃЏрЃърЃћрЃбрЃўрЃфрЃўрЃўрЃА - "Walmart Recruiting - Store Sales Forecasting" - рЃАрЃЮрЃџрЃБрЃерЃћрЃюрЃА. рЃЎрЃЮрЃюрЃЎрЃБрЃарЃАрЃўрЃАрЃљ рЃЊрЃљ time series рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃЮ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљ рЃўрЃАрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЏрЃљрЃдрЃљрЃќрЃўрЃљрЃерЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃЊрЃћрЃърЃљрЃарЃбрЃљрЃЏрЃћрЃюрЃбрЃўрЃљ рЃЊрЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЊрЃћрЃърЃљрЃарЃбрЃљрЃЏрЃћрЃюрЃбрЃўрЃА рЃЎрЃЋрЃўрЃарЃћрЃБрЃџрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљ. рЃњрЃљрЃЏрЃЮрЃгрЃЋрЃћрЃЋрЃљрЃА рЃќрЃарЃЊрЃўрЃА рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃцрЃљрЃАрЃЊрЃљрЃЎрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА (markdown) рЃдрЃЮрЃюрЃўрЃАрЃФрЃўрЃћрЃЉрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃњрЃљрЃЋрЃџрЃћрЃюрЃљрЃА рЃљрЃ«рЃЊрЃћрЃюрЃА рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃќрЃћ.

## рЃЕрЃЋрЃћрЃюрЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃГрЃарЃћрЃџрЃљрЃЊ

рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃЋрЃфрЃљрЃЊрЃћрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў:
- **рЃЎрЃџрЃљрЃАрЃўрЃЎрЃБрЃарЃў рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў**: ARIMA, SARIMA, Prophet
- **Tree-Based рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў**: LightGBM, XGBoost
- **Deep Learning рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў**: N-BEATS, D-LINEAR, PatchTST

### рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:
- **data_exploration.py** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃў
- **data_preprocessing_pipeline.py** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃА рЃърЃљрЃўрЃърЃџрЃљрЃўрЃюрЃў
- **model_experiment_*.py** - рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃА рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў

## ­ЪћД Feature Engineering

### рЃЊрЃарЃЮрЃўрЃЌрЃў рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў:
- рЃгрЃћрЃџрЃў, рЃЌрЃЋрЃћ, рЃЎрЃЋрЃўрЃарЃљ, рЃЊрЃдрЃћ
- рЃЎрЃЋрЃљрЃарЃбрЃљрЃџрЃў, рЃЎрЃЋрЃўрЃарЃўрЃА рЃЊрЃдрЃћ, рЃгрЃџрЃўрЃА рЃЊрЃдрЃћ
- рЃфрЃўрЃЎрЃџрЃБрЃарЃў рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў (sin/cos рЃбрЃарЃљрЃюрЃАрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃћрЃЉрЃў)
- рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЎрЃЋрЃўрЃарЃћрЃЉрЃљрЃЏрЃЊрЃћ рЃЊрЃљрЃарЃЕрЃћрЃюрЃўрЃџрЃў рЃЊрЃарЃЮ

### Lag рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў:
- рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА lag рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў: 1, 2, 3, 4, 8, 52 рЃЎрЃЋрЃўрЃарЃљ
- рЃЏрЃЮрЃФрЃарЃљрЃЋрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃЮрЃћрЃЉрЃў: 4, 8, 12 рЃЎрЃЋрЃўрЃарЃўрЃА рЃцрЃљрЃюрЃ»рЃарЃћрЃЉрЃў
- рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃ«рЃарЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃљрЃюрЃ»рЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

### рЃЏрЃљрЃдрЃљрЃќрЃўрЃљ/рЃЊрЃћрЃърЃљрЃарЃбрЃљрЃЏрЃћрЃюрЃбрЃўрЃА рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў:
- рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃќрЃЮрЃЏрЃљ рЃЊрЃљ рЃбрЃўрЃърЃў (A, B, C)
- рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃћрЃърЃљрЃарЃбрЃљрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃў
- рЃЋрЃљрЃарЃўрЃљрЃфрЃўрЃўрЃА рЃЎрЃЮрЃћрЃцрЃўрЃфрЃўрЃћрЃюрЃбрЃў (CV)

### рЃћрЃЎрЃЮрЃюрЃЮрЃЏрЃўрЃЎрЃБрЃарЃў рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў:
- рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃў
- рЃАрЃљрЃгрЃЋрЃљрЃЋрЃўрЃА рЃцрЃљрЃАрЃў, CPI, рЃБрЃЏрЃБрЃерЃћрЃЋрЃарЃЮрЃЉрЃљ
- рЃћрЃЎрЃЮрЃюрЃЮрЃЏрЃўрЃЎрЃБрЃарЃў рЃАрЃбрЃарЃћрЃАрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў

### Markdown рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў:
- рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў markdown-рЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљ
- рЃљрЃЦрЃбрЃўрЃБрЃарЃў markdown-рЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ
- рЃ»рЃљрЃЏрЃБрЃарЃў markdown рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ

## рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃБрЃџрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃерЃў рЃњрЃљрЃЊрЃљрЃДрЃЋрЃљрЃюрЃљ

- **рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃбрЃўрЃърЃў**: One-hot encoding (Is_TypeA, Is_TypeB, Is_TypeC)
- **рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃќрЃЮрЃЏрЃљ**: Binning рЃЊрЃљ Label Encoding (Small, Medium, Large, Extra_Large)
- **рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЊрЃарЃЮрЃерЃљ**: рЃЉрЃўрЃюрЃљрЃарЃБрЃџрЃў рЃфрЃЋрЃџрЃљрЃЊрЃў (0/1)

## NaN рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ

### Markdown рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў:
- NaN Рєњ 0 (рЃцрЃљрЃАрЃЊрЃљрЃЎрЃџрЃћрЃЉрЃўрЃА рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљ)
- рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў рЃЏрЃљрЃарЃЎрЃЊрЃљрЃБрЃюрЃћрЃЉрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

### рЃћрЃЎрЃЮрЃюрЃЮрЃЏрЃўрЃЎрЃБрЃарЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў:
- рЃЏрЃћрЃЊрЃўрЃљрЃюрЃўрЃЌ рЃерЃћрЃЋрЃАрЃћрЃЉрЃљ (Temperature, Fuel_Price, CPI, Unemployment)

### Lag рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў:
- NaN рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃАрЃљрЃбрЃћрЃАрЃбрЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃерЃў (рЃўрЃАрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљ)

## Cleaning рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃћрЃЉрЃў

1. **рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ**: рЃљрЃЉрЃАрЃЮрЃџрЃБрЃбрЃБрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃА рЃљрЃдрЃћрЃЉрЃљ
2. **рЃюрЃБрЃџрЃЮрЃЋрЃљрЃюрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃљ**: рЃџрЃћрЃњрЃўрЃбрЃўрЃЏрЃБрЃарЃў рЃюрЃБрЃџрЃЮрЃЋрЃљрЃюрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃў
3. **рЃЌрЃљрЃарЃўрЃдрЃћрЃЉрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ**: pd.to_datetime рЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃљрЃфрЃўрЃљ
4. **рЃЊрЃБрЃЉрЃџрЃўрЃЎрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ**: Store-Dept-Date рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃЮрЃЉрЃљ

## Feature Selection

### рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃћрЃЉрЃў:

1. **рЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў**:
   - рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃЌрЃљрЃю рЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃџрЃўрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў рЃцрЃўрЃЕрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА
   - рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃЊрЃљ CPI рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЊрЃљрЃЊрЃћрЃЉрЃўрЃЌ рЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃљрЃА

2. **рЃЊрЃЮрЃЏрЃћрЃўрЃюрЃўрЃА рЃфрЃЮрЃЊрЃюрЃљрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃарЃЕрЃћрЃЋрЃљ**:
   - рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃърЃћрЃарЃўрЃЮрЃЊрЃћрЃЉрЃў (Super Bowl, Thanksgiving, Christmas)
   - рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃў рЃърЃљрЃбрЃћрЃарЃюрЃћрЃЉрЃў (52 рЃЎрЃЋрЃўрЃарЃўрЃљрЃюрЃў рЃфрЃўрЃЎрЃџрЃў)

3. **Lag рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљ**:
   - ARIMA/SARIMA рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА lag рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃБрЃџрЃўрЃљ
   - Prophet рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЊрЃарЃЮрЃўрЃЌ рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃА

## рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў

рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃљ рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃў, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА D-LINEAR, N-BEATS, PatchTST, ARIMA, SARIMA, Prophet, LightGBM, рЃЊрЃљ XGBoost.

рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ рЃЕрЃљрЃбрЃљрЃарЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃюрЃљрЃЎрЃарЃћрЃЉрЃќрЃћ. рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЏрЃћрЃбрЃарЃўрЃЎрЃћрЃЉрЃўрЃАрЃљ (MAE, RMSE рЃЊрЃљ рЃљ.рЃе.) рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ, рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃўрЃ«рЃўрЃџрЃЮрЃЌ рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃљрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃюрЃЮрЃБрЃЌрЃЉрЃБрЃЦрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ `notebooks/` рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў. рЃћрЃА рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃюрЃЮрЃБрЃЌрЃЉрЃБрЃЦрЃћрЃЉрЃў рЃарЃЕрЃћрЃЉрЃљ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃљрЃЊрЃћрЃарЃЌ рЃЊрЃљ рЃБрЃбрЃДрЃБрЃљрЃа рЃгрЃДрЃљрЃарЃЮрЃЊ.

## рЃЎрЃџрЃљрЃАрЃўрЃЎрЃБрЃарЃў рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў

### ARIMA
- **рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ**: `statsmodels`
- **рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ**: ARIMA рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЊрЃћрЃърЃљрЃарЃбрЃљрЃЏрЃћрЃюрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЊрЃарЃЮрЃўрЃА рЃАрЃћрЃарЃўрЃћрЃЉрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА - рЃљрЃЋрЃбрЃЮрЃарЃћрЃњрЃарЃћрЃАрЃўрЃљ (AR), рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў (I) рЃЊрЃљ рЃЏрЃЮрЃФрЃарЃљрЃЋрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃЮ (MA) - рЃљрЃдрЃАрЃљрЃгрЃћрЃарЃљрЃЊ. рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃњрЃљрЃгрЃЋрЃарЃЌрЃюрЃўрЃџрЃў рЃўрЃЦрЃюрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃю рЃљрЃњрЃарЃћрЃњрЃўрЃарЃћрЃЉрЃБрЃџ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ.
- **рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў**: рЃЮрЃърЃбрЃўрЃЏрЃљрЃџрЃБрЃарЃў $(p, d, q)$ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў рЃюрЃљрЃърЃЮрЃЋрЃюрЃў рЃўрЃЦрЃюрЃљ `pmdarima`-рЃА `auto_arima` рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ AIC (Akaike Information Criterion) рЃЊрЃљ BIC (Bayesian Information Criterion) рЃЎрЃарЃўрЃбрЃћрЃарЃўрЃБрЃЏрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃЌ. `model_experiment_ARIMA.ipynb` рЃюрЃЮрЃБрЃЌрЃЉрЃБрЃЦрЃерЃў, `mlflow.log_param("pipeline_type", "hierarchical_forecast")` рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА рЃўрЃћрЃарЃљрЃарЃЦрЃўрЃБрЃџ рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљрЃќрЃћ, рЃАрЃљрЃЊрЃљрЃф рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃћрЃЉрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃћрЃърЃљрЃарЃбрЃљрЃЏрЃћрЃюрЃбрЃўрЃА рЃЊрЃЮрЃюрЃћрЃќрЃћ.
- **рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ**:

    - **MSE**: $251,577,419,003.5518$
    - **RMSE**: $501,574.9366$
    - **MAPE**: $12.3789$
    - **AIC**: $2822.4276$
    - **BIC**: $2831.6366$
- **рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃћрЃЉрЃў**: ARIMA рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮ рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃў рЃЊрЃарЃЮрЃўрЃА рЃАрЃћрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃўрЃАрЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ рЃерЃћрЃцрЃљрЃАрЃЊрЃљ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃбрЃарЃўрЃЎрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў рЃўрЃЦрЃюрЃљ MLflow-рЃќрЃћ.

### SARIMA
- **рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ**: `statsmodels`
- **рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ**: SARIMA (Seasonal ARIMA) рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃў рЃърЃљрЃбрЃћрЃарЃюрЃћрЃЉрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃЊрЃарЃЮрЃўрЃА рЃАрЃћрЃарЃўрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ Walmart-рЃўрЃА рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃљрЃЦрЃЋрЃЌ рЃЏрЃЎрЃљрЃцрЃўрЃЮ рЃДрЃЮрЃЋрЃћрЃџрЃЎрЃЋрЃўрЃарЃћрЃБрЃџрЃў рЃЊрЃљ рЃДрЃЮрЃЋрЃћрЃџрЃгрЃџрЃўрЃБрЃарЃў рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃљ. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЊрЃћрЃърЃљрЃарЃбрЃљрЃЏрЃћрЃюрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃў рЃЋрЃљрЃарЃўрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃЊрЃћрЃърЃљрЃарЃбрЃљрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃфрЃљрЃџрЃЎрЃћ рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў рЃњрЃљрЃгрЃЋрЃарЃЌрЃюрЃљ.

- **рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў**: рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃљ Optuna-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃљ `num_leaves`, `learning_rate`, `feature_fraction`, `bagging_fraction` рЃЊрЃљ рЃарЃћрЃњрЃБрЃџрЃљрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃарЃЏрЃўрЃюрЃћрЃЉрЃў (`reg_alpha`, `reg_lambda`).
- **рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ**: рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃЏрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃЏрЃўрЃљрЃдрЃгрЃўрЃљ cross-validation MAE-рЃА рЃЊрЃљрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃўрЃЌ **$1,135**.
- **рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃћрЃЉрЃў**: рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљрЃЏ Optuna-рЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃюрЃљрЃЊ рЃњрЃљрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃљ рЃерЃћрЃЊрЃћрЃњрЃў рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃЏрЃЮрЃЊрЃћрЃџрЃЌрЃљрЃю рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ. рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃљ, рЃарЃЮрЃЏ `Store`, `Dept` рЃЊрЃљ рЃЌрЃљрЃарЃўрЃдрЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃњрЃљрЃЋрЃџрЃћрЃюрЃўрЃљрЃюрЃў рЃърЃарЃћрЃЊрЃўрЃЦрЃбрЃЮрЃарЃћрЃЉрЃў рЃўрЃДрЃЮ.

### XGBoost
- **рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ**: `xgboost`
- **рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃА рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃўрЃљ**: рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃА рЃърЃљрЃўрЃърЃџрЃљрЃўрЃюрЃў, рЃарЃљрЃф LightGBM-рЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃљрЃЏрЃљрЃф рЃюрЃћрЃЊрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃњрЃљрЃЊрЃљрЃљрЃЦрЃфрЃўрЃљ рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃЊрЃўрЃЊрЃљрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃЊ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃАрЃљрЃцрЃћрЃарЃўрЃАрЃўрЃљ рЃ«рЃћрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
- **рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў**: рЃЊрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЊрЃљ Optuna-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, рЃљрЃЦрЃфрЃћрЃюрЃбрЃўрЃЌ `n_estimators`, `max_depth`, `learning_rate`, `subsample` рЃЊрЃљ `colsample_bytree`-рЃќрЃћ.
- **рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ**: рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃљ cross-validation Weighted MAE (WMAE) рЃЊрЃљрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃўрЃЌ **$2,450$.**
- **рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃћрЃЉрЃў**: XGBoost-рЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃљ рЃФрЃџрЃўрЃћрЃарЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ, SHAP рЃљрЃюрЃљрЃџрЃўрЃќрЃЏрЃљ рЃЎрЃў рЃЏрЃЮрЃњрЃЋрЃфрЃљ рЃдрЃарЃЏрЃљ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃљ рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃА рЃгрЃЋрЃџрЃўрЃџрЃќрЃћ. `Store`, `Dept` рЃЊрЃљ `Size` рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃЊрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў.

## рЃдрЃарЃЏрЃљ рЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў

### D-LINEAR
- **рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ**: `torch`
- **рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃА рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃўрЃљ**: Lag рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃБрЃџ рЃўрЃЦрЃюрЃљ. рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃЊрЃљрЃљрЃЏрЃБрЃерЃљрЃЋрЃљ рЃЊрЃарЃЮрЃўрЃЌрЃў рЃЏрЃгрЃЎрЃарЃўрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃбрЃарЃћрЃюрЃЊрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃљрЃЊ рЃЊрЃљрЃерЃџрЃўрЃА рЃњрЃќрЃўрЃЌ. рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃЕрЃљрЃбрЃљрЃарЃЊрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃБрЃюрЃўрЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃБрЃџ, рЃўрЃАрЃћ рЃЏрЃБрЃџрЃбрЃўрЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃБрЃџ рЃарЃћрЃЪрЃўрЃЏрЃћрЃЉрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃћрЃА рЃБрЃЎрЃљрЃюрЃљрЃАрЃЎрЃюрЃћрЃџрЃў рЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃЊрЃљ `Temperature`, `Fuel_Price`, `CPI`, `Unemployment` рЃЊрЃљ `IsHoliday` рЃњрЃљрЃарЃћ рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃА.
- **рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў**: рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃЏ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃљ 52-рЃўрЃА рЃбрЃЮрЃџрЃў рЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃћ, 8-рЃўрЃА рЃбрЃЮрЃџрЃў рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃћ рЃЊрЃљ 25-рЃўрЃА рЃбрЃЮрЃџрЃў рЃЏрЃЮрЃФрЃарЃљрЃЋрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃЮрЃА рЃЉрЃўрЃарЃЌрЃЋрЃў.
- **рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ**: рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃБрЃюрЃўрЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃБрЃџрЃЏрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃЏрЃўрЃљрЃдрЃгрЃўрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА MAE-рЃА **$2.8M** рЃЏрЃЌрЃџрЃўрЃљрЃю рЃљрЃњрЃарЃћрЃњрЃўрЃарЃћрЃЉрЃБрЃџ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ.
- **рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃћрЃЉрЃў**: D-LINEAR-рЃўрЃА рЃЊрЃљрЃерЃџрЃўрЃА рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃљрЃдрЃЏрЃЮрЃЕрЃюрЃЊрЃљ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃарЃБрЃџрЃў рЃърЃљрЃбрЃћрЃарЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃГрЃћрЃарЃљрЃЊ. рЃбрЃарЃћрЃюрЃЊрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃЏрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃЏрЃљ рЃгрЃарЃцрЃўрЃЋрЃЏрЃљ рЃцрЃћрЃюрЃћрЃЉрЃЏрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮ рЃФрЃџрЃўрЃћрЃарЃў рЃАрЃљрЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃЮ рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃЏрЃЮрЃЊрЃћрЃџрЃў.

### N-BEATS
- **рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ**: `darts` (backend: `pytorch-lightning`)
- **рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃА рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃўрЃљ**: рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃЊрЃљ рЃљрЃњрЃарЃћрЃњрЃўрЃарЃћрЃЉрЃБрЃџ рЃЏрЃЌрЃџрЃўрЃљрЃю рЃЎрЃЋрЃўрЃарЃћрЃБрЃџ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃўрЃЌ рЃЏрЃгрЃЎрЃарЃўрЃЋрЃќрЃћ, lag рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ. `darts` рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃљ `TimeSeries` рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ рЃЊрЃљ рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ.
- **рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў**: рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў рЃЏрЃўрЃдрЃгрЃћрЃБрЃџ рЃўрЃЦрЃюрЃљ рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃўрЃЌ (`generic_architecture=False`), рЃбрЃарЃћрЃюрЃЊрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА 2 рЃАрЃбрЃћрЃЎрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, рЃбрЃарЃћрЃюрЃЊрЃўрЃА рЃърЃЮрЃџрЃўрЃюрЃЮрЃЏрЃўрЃБрЃарЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃЌ 2 рЃЊрЃљ рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃўрЃЌ 24.
- **рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ**: рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃўрЃарЃћрЃЉрЃљрЃЊрЃЏрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃЏрЃўрЃљрЃдрЃгрЃўрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА MAE-рЃА рЃЊрЃљрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃўрЃЌ **$2.1M** рЃЏрЃЌрЃџрЃўрЃљрЃю рЃљрЃњрЃарЃћрЃњрЃўрЃарЃћрЃЉрЃБрЃџ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ.
- **рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃћрЃЉрЃў**: N-BEATS-рЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃљ рЃФрЃџрЃўрЃћрЃарЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ рЃЊрЃљ рЃЏрЃўрЃАрЃЏрЃљ рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃўрЃарЃћрЃЉрЃљрЃЊрЃЏрЃљ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљрЃЏ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљ рЃЏрЃЮрЃњрЃЋрЃфрЃљ рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃў рЃЊрЃљрЃњрЃЋрЃћрЃерЃљрЃџрЃљ рЃбрЃарЃћрЃюрЃЊрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃљрЃЊ, рЃарЃљрЃф рЃдрЃўрЃарЃћрЃЉрЃБрЃџ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃњрЃЋрЃљрЃгрЃЋрЃЊрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.

### PatchTST
- **рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ**: `torch`
- **рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃўрЃА рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃўрЃљ**: рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃЊрЃљрЃљрЃЏрЃБрЃерЃљрЃЋрЃљ рЃЊрЃарЃЮрЃўрЃЌрЃў рЃЏрЃгрЃЎрЃарЃўрЃЋрЃў рЃЏрЃўрЃАрЃў рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљрЃЊ (patch-рЃћрЃЉрЃљрЃЊ) рЃЊрЃљрЃДрЃЮрЃцрЃўрЃА рЃњрЃќрЃўрЃЌ. рЃћрЃА рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА Transformer рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљрЃА рЃцрЃЮрЃЎрЃБрЃАрЃўрЃарЃћрЃЉрЃљ рЃЏрЃЮрЃљрЃ«рЃЊрЃўрЃюрЃЮрЃА рЃџрЃЮрЃЎрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃўрЃЌ рЃърЃљрЃбрЃћрЃарЃюрЃћрЃЉрЃќрЃћ. Lag рЃцрЃўрЃЕрЃћрЃарЃћрЃЉрЃў рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃБрЃџ рЃўрЃЦрЃюрЃљ.
- **рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў**: Grid search-рЃЏрЃљ рЃњрЃљрЃЏрЃЮрЃљрЃЋрЃџрЃўрЃюрЃљ рЃЮрЃърЃбрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ: `patch_len` 8, `stride` 8, `d_model` 128, `n_heads` 8 рЃЊрЃљ `num_layers` 4.
- **рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ**: рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃЌ рЃњрЃљрЃгрЃЋрЃарЃЌрЃюрЃўрЃџрЃЏрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃЏрЃўрЃљрЃдрЃгрЃўрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА MAE-рЃА **$0.08$** рЃАрЃЎрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ.
- **рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃћрЃЉрЃў**: PatchTST-рЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃў рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃљ рЃЊрЃарЃЮрЃўрЃЌрЃў рЃЏрЃгрЃЎрЃарЃўрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃерЃў рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃўрЃА рЃњрЃќрЃўрЃЌ. Transformer-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ рЃЏрЃФрЃџрЃљрЃЋрЃарЃў рЃљрЃдрЃЏрЃЮрЃЕрЃюрЃЊрЃљ рЃљрЃЏ рЃАрЃљрЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃЮ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА.
